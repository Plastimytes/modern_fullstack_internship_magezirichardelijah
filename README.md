**STUDENT TASK MANAGER**

An application designed to help students organise their academic lifestyle.

*Features*


User registration and login. Super Admin, Admin user and client.

Create and manage tasks.

Super Admin has overview on the whole application, How its being used, spikes in usage and all users clienst and Admins alike.

Admin can just veiw the users.

*Tech Stack*

Frontend : React, TypeScript, Tailwind CSS

Backend : Spring Boot, Java

Database : MariaDB STRICTLY
